# InmoVox AI — Landing (Sofía, agente de voz IA para inmobiliarias)

Sitio estático (HTML/CSS puro, sin build). Landing de lista de espera pre-lanzamiento
con páginas de Kit Digital, ciudades (Madrid, Barcelona, Valencia) y blog.
Formularios conectados a Supabase (tabla `public.waitlist_signups`).

Pendiente antes del lanzamiento: sustituir `TU-DOMINIO-AQUI.com` por el dominio real
en canonical/og de las 11 páginas y en `sitemap.xml`, y añadir `og:image` (1200x630).
